Magento:

1. Admin side:
    A. Create Integration:
     System -> Integrations -> Add New Integration
     API -> Available APIs: Resource Access: All
     Save & Activate
     Get Access Token from Integration Details

    B. Create New User Role & User:
     System -> User Roles -> Add New Role
     Role Resources: Resource Access -> All

     System -> All Users -> Add New User (username + password)
     Assign New User Role to User

2. API:
Request the access token:
    curl -X POST "http://127.0.0.1:32769/index.php/rest/V1/integration/admin/token" -H "Content-Type:application/json"  -d '{"username":"test", "password":"test123"}'
Get all customers:
    curl -X GET "http://127.0.0.1:32769/index.php/rest/V1/customers/search?searchCriteria" -H "Authorization: Bearer a1fwdk8qi06cyo9v855e21wj8q0etr0o"
Get all orders:
    curl -X GET "http://127.0.0.1:32769/index.php/rest/V1/orders?searchCriteria" -H "Authorization: Bearer a1fwdk8qi06cyo9v855e21wj8q0etr0o"

X2CRM:
1. Admin side:
-Create User:
    Users -> Create User: fill API Key field
    Update User: correct API Key
-Create User Role:
    Admin -> User Management -> Manage Roles -> Add Role:
        -add new user to role
        -Add all on View Permissions
-Edit Role Access(again):
    Admin -> User Management -> Edit User Permissions & Access Rules
        -select new role, check mark on View

2. API:
-Check user permissions setup with curl:
    curl -X GET "{domain_name}/index.php/api2/Accounts/" -H "Authorization: Basic {access_key}"
    curl -X GET "{domain_name}/index.php/api2/Contacts/" -H "Authorization: Basic {access_key}"
    access_key is "username:api_key" in Base-64 encoding

3. Documentation:
http://wiki.x2crm.com/wiki/REST_API_Reference

Zoho:
1. Admin Side:
    Generate authtoken:
    Tools -> Setup -> EXTENSIONS & APIs -> APIs -> Settings -> Authentication Token Generation
    -> Browser Mode (INTERNAL USE) -> Generate

2. API:
Get all accounts:
    curl -X GET "https://crm.zoho.eu/crm/private/json/Accounts/getRecords"
    -H "Authorization: Zoho-authtoken {authtoken}"

Get all contacts:
    curl -X GET "https://crm.zoho.eu/crm/private/json/Contacts/getRecords"
    -H "Authorization: Zoho-authtoken {authtoken}"

3. Documentation:
    https://www.zoho.eu/crm/help/api/welcome.html

4. Notes:
    params = {'scope':'crmapi', 'fromIndex': 1, 'toIndex': 200, 'newFormat': 2, 'version': 2}

    -fromIndex -> toIndex defaults to 1-20 (max=200)
    -newFormat=1:To exclude fields with "null" values while inserting data from your CRM account.
    -newFormat=2:To include fields with "null" values while inserting data from your CRM account.

    Max 200 records per request
    Blind pagination: no api call for records count
    No more data to show:
        {'response': {'nodata': {'code': '4422', 'message': 'There is no data to show'}, 'uri': '/crm/private/json/Accounts/getRecords'}}

    Daily API requests limit based on subscription type:
        https://www.zoho.eu/crm/help/api/api-limits.html
